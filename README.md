# Proyecto #3 de Algoritmos


# **==Instrucciones para iniciar el proyecto==**

1) make folders
2) make
3) make run
4) make clean (cada que se cambie algo de codigo).

# **==Modo de trabajo==**

1) Fork de este repositorio.
2) Avanzar cada integrante su parte, cada integrante trabaja en su repositorio forkeado de este mismo.
3) Comentar lo mas claro posible para facilitar el trabajo de todos.

- Siempre asegurarse de tener el fork actualizado, para evitar conflictos de merge.

# **==Modo de subir trabajo==**

1) make clean
2) git push al repositorio forkeado
3) hacer un pull request de los cambios para hacer merge del forkeado y el repositorio base (se puede realizar desde github en los repositorios forkeados).
